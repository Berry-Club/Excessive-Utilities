# Flight

Excessive Utilities has three main flight or fall-control items. They all use Grid Power only while their effect is active.

## Chicken Wing Ring

The **Chicken Wing Ring** slows your fall while you hold jump in the air.

| Default | Value |
| --- | ---: |
| GP cost | 1 |
| Use time | 10 seconds |
| Recharge time | 5 seconds |
| Max fall speed | 0.1 blocks/tick |

It is useful as early fall protection. It does not provide upward movement.

## Ring of the Flying Squid

The **Ring of the Flying Squid** pushes you upward while you hold jump in the air, as long as you are not already flying.

| Default | Value |
| --- | ---: |
| GP cost | 16 |
| Use time | 15 seconds |
| Recharge time | 10 seconds |
| Max upward speed | 1 block/tick |

It recharges while inactive. It is a burst-mobility item, not permanent creative flight.

## Angel Ring

The **Angel Ring** grants creative-style flight while equipped as a Curio. It costs **64 GP** by default while you are flying. Creative-mode players do not pay this GP cost.

If your grid is overloaded, the Angel Ring removes its flight effect until enough GP is available again.

Angel Rings can have different wing appearances:

- Invisible
- Feather
- Butterfly
- Demon
- Gold
- Bat

The wing type changes visuals, not the GP cost.

## Troubleshooting

If a ring does nothing, check these first:

- Make sure your GP usage is not above your GP capacity.
- Make sure the ring has recharged if it is the Chicken Wing Ring or Ring of the Flying Squid.
- Make sure you are holding jump and are not on the ground for the temporary rings.
- Use the Power Manager to check whether another consumer is occupying your GP.

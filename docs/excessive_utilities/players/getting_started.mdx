# Getting Started

Excessive Utilities has two kinds of documentation in this wiki:

- **For Players** pages explain larger systems and common workflows.
- **Content** pages list individual blocks, items, enchantments, recipes, and obtaining info.

If you are new to the mod, start by learning **Grid Power** and a few utility items. Many stronger tools and machines either reserve GP directly or rely on items made in GP-powered machines.

## Early Priorities

1. Make a **Manual Mill** for your first Grid Power.
2. Make a **Power Manager** so you can see what is producing and consuming GP.
3. Make a **Resonator** once recipes start asking for resonated materials.
4. Add environmental mills when you want passive GP.

The Manual Mill is the simplest starting producer because it only needs someone to crank it. Passive mills are better for a base, but they each have placement requirements.

## Useful Early Utilities

| Item or Block | Why it matters |
| --- | --- |
| Power Manager | Lists your active GP producers and consumers. |
| Manual Mill | Provides early GP without a setup. |
| Resonator | Crafts GP-gated materials and components. |
| Transfer Nodes and Pipes | Move items, fluids, or energy around compactly. |
| Magnum Torch or Chandelier | Prevent hostile spawns in an area. |
| Golden Lasso | Moves passive mobs. |

Most individual content pages include an obtaining section, so use the Content section when you need exact recipes.

## Progression Notes

GP is per-player, so placing GP blocks yourself matters. A machine placed by one player draws from that player's grid, even if another player uses the block later.

Speed Upgrades are powerful, but they add GP cost. If a machine suddenly stops after you add upgrades, check the Power Manager and make sure your grid is not overloaded.

Flight starts with temporary rings and eventually the Angel Ring. The temporary rings only draw GP while actively used; the Angel Ring draws GP while you are flying.

A **Creative Upgrade** can be placed inside [Transfer](@excessive_utilities:transfer_nodes) and [Retrieval Nodes](@excessive_utilities:retrieval_nodes) to prevent them from emptying their buffer, which allows them to output indefinitely without needing to be refilled.

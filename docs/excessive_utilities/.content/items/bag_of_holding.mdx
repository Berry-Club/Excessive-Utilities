The **Bag of Holding** is a portable inventory that stores as much as a double chest. Simply right-click to open the bag's inventory.

Each Bag has a UUID, which is a pointer to the Bag's inventory. If you lose the Bag, you can just spawn a new one in with the same UUID if needed.

The **Erosion Shovel** instantly mines "filler" blocks, but also voids all of their drops. Additionally, it slightly increases your block reach, and has infinite durability.

It cannot be used to mine non-filler blocks at all.

Filler blocks are defined as any block that has the block tag `#excessive_utilities:erosion_shovel_target`.

By default, this only includes the following blocks:
- `#minecraft:dirt`
- `#c:sands`,
- `#c:gravels`

The **Destruction Pickaxe** instantly mines "filler" blocks, but also voids all of their drops. Additionally, it slightly increases your block reach.

It cannot be used to mine non-filler blocks at all.

Filler blocks are defined as any block that has the block tag `#excessive_utilities:destruction_pickaxe_target`.

By default, this only includes the following blocks:
- `#c:stones`
- `#c:netherracks`

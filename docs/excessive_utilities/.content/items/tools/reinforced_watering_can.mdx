The **Reinforced Watering Can** is an upgraded version of the [Watering Can](@excessive_utilities:watering_can).

It functions the same, but has a wider radius, works faster, and can be used by Fake Players without breaking.

## Configuration

The amount of Water that it drains per tick is configurable, and can be set to 0 to make it free.

The radius it waters and the speed that it forces ticks can also be configured.

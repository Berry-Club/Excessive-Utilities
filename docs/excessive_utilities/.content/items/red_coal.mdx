**Red Coal** is an improved Furnace fuel that burns longer than regular coal, with an additional cost.

Red Coal is owned by the player that held it last, and uses [Grid Power]($players/grid_power) when burned in a Furnace.

By default, it costs 4.0 GP to apply its bonus, and it burns for 8 times as long as regular Coal. If you can't afford the GP cost, it will burn for the same amount of time as regular Coal, without consuming any GP.

<Callout>
For technical reasons, all special mechanics only function when the Red Coal is burned in Furnaces, or other block entities that extend the `AbstractFurnaceBlockEntity` class.
	</Callout>

## Obtaining

Red Coal can be obtained by passing Coal through a [Resonator](@excessive_utilities:resonator), which costs 16.0 GP.

## Configuration

The GP cost and the burn time multiplier can be configured.

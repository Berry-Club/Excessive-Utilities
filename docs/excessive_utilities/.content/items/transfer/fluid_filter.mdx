**Fluid Filter** is used to filter what fluids are allowed in certain things, like [Fluid Transfer Nodes](@excessive_utilities:transfer_nodes).

Opening the Filter allows you to insert up to 16 fluid-carrying items. By default, these are the fluids that will pass the filter. Putting another Fluid Filter inside will nest it, allowing you to combine multiple Filters in one item!

There are three buttons along the bottom:

- Invert
- Use Tags
- Ignore All Components

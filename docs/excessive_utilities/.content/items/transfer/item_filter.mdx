**Item Filter** is used to filter what items are allowed in certain things, like [Transfer Nodes](@excessive_utilities:transfer_nodes).

Opening the Filter allows you to insert up to 16 items. By default, these are the items that will pass the filter. Putting another Item Filter inside will nest it, allowing you to combine multiple Filters in one item!

There are four buttons along the bottom:

- Invert
- Use Tags
- Ignore Damage
- Ignore All Components

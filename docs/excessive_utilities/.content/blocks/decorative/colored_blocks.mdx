**Colored Blocks** are decorative color variants of common building blocks in Excessive Utilities.

Each family is available in all 16 dye colors:

- Colored Bricks
- Colored Coal Blocks
- Colored Cobblestone
- Colored Glowstone
- Colored Lapis Blocks
- Colored Obsidian
- Colored Planks
- Colored Quartz Blocks
- Colored Redstone Blocks
- Colored Redstone Lamps
- Colored Soul Sand
- Colored Stone Bricks
- Colored Stone
- Colored Wood
- Lapis Caelestis

These blocks share the behavior of their base block family and are documented together to avoid one page per color.

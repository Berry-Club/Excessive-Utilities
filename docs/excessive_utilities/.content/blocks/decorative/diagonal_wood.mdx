**Diagonal Wood** is a decorative block from Excessive Utilities.

This block supports connected textures when [Athena](https://www.curseforge.com/minecraft/mc-mods/athena) is installed. Athena is also available on [Modrinth](https://modrinth.com/mod/athena-ctm).

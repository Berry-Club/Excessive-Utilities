The **Overclocked Generator** burns furnace fuel to produce FE. Fuel will burn VERY fast, producing all of the FE at once.

Fuels will be burned out immediately, lasting only a single tick. Each tick that it would have burned in a Furnace will contribute 40 FE.

## Configuration

The amount of FE per Furnace fuel tick can be configured.

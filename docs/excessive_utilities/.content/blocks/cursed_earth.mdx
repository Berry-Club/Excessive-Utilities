**Cursed Earth** will rapidly spawn monsters on top of itself. If it lights on fire, it will spread EXTREMELY QUICKLY to other nearby Cursed Earth blocks, burning them into Dirt.

Cursed Earth can be obtained by using a [Drop of Evil](@excessive_utilities:drop_of_evil) on Dirt blocks.

## Configuration

The following can be configured:
- The number of ticks between each attempt to do work
- The chance per work tick that a mob will spawn
- The max amount of nearby mobs that can be present before it stops spawning more
- The radius in which it checks for nearby mobs

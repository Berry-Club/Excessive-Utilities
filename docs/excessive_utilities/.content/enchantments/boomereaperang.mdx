**Boomereaperang** is an enchantment for the [Magical Boomerang](@excessive_utilities:magical_boomerang) that causes the Boomerang to break certain blocks in its path.

It breaks any blocks in the `#excessive_utilities:collectable_by_boomereaperang` tag, which by default includes:
- `#minecraft:flowers`
- Short Grass
- Tall Grass
- Ferns
- Large Ferns

The radius is configurable.
